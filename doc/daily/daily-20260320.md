**設計・実装**: `doc/design-background-customization-20250320.md` に基づき実装。

- **ユーザー選択**: 複数の背景候補からユーザーが選択可能
- **テーマ拡張**: `backgroundDisplay`、`defaultBackgroundPresetId` をテーマプリセットに追加（宣言的設計）

### 2. 必須ドキュメント整理・ARCHITECTURE拡充・README更新

- **README**: 設計・計画・タスク管理のAI提案を明記し、現状齟齬を修正
- **.cursorrules**: 開発開始手順を簡素化

### 3. 本番ビルドでのデバッグログ抑制

- **不具合修正**: `devLog` / `logger` において、ブラウザで `process` 未定義のときに `getAppEnv` が常に `development` になる問題を修正
- **対応**: `process.env.*` を直接参照し、Vite の `define` と連携
- **vite.config**: `loadEnv` + `define` で `NODE_ENV` / `VITE_LOG_LEVEL` / `VITE_DEBUG_LOGS` を埋め込み
- **コード置換**: 主要な `console.log` を `devLog` に置換、logger テスト拡張
- **bug-feature-memo**: 完了タスクを反映

### 4. アイコン差し替え・PWAアイコン統一
